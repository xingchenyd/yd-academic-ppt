# Academic template fidelity

Use this reference for every default academic-editorial deck. The outer frame is stable; the center adapts to the topic. The goal is recognizably the same editorial family on version 1, not merely a similar palette.

## 1. What creates the identity

The reference style is identified, in order, by:

1. substantial navy top-left banner with angled end and white identity text;
2. full-width navy footer with a red angled right tab and page/section text;
3. red section eyebrow plus navy descriptor;
4. compact navy number badge joined visually to a large navy serif title;
5. short thick red underline beneath the title start;
6. warm off-white canvas, rounded bordered modules, navy/red headers, circular icons;
7. compact editorial tables, charts, formulas, and takeaway strips;
8. dense but orderly information architecture.

If items 1–5 are absent, the result is not a faithful YD academic page even if the colors are correct.

## 2. Reference geometry for a 13.333 × 7.5 in slide

Use these as starting ranges, then adjust for the title and content:

| Element | Recommended geometry |
| --- | --- |
| Top-left navy banner | x 0, y 0, w 4.9–5.7, h 0.52–0.68 in; angled/clipped right edge |
| Top-right traits | x 9.8–12.9, y 0.20–0.42 in |
| Section eyebrow | x 0.50–0.62, y 0.72–0.92 in |
| Number badge | x 0.50–0.62, y 1.02–1.28, w/h 0.38–0.48 in |
| Main title | x 1.02–1.18, y 0.98–1.25, up to x 12.85 |
| Red underline | x 0.50–0.62, directly below title; w 0.45–1.10, h 0.035–0.065 in |
| Subtitle | x 0.50–0.62, below underline, 0.22–0.38 in high |
| Content zone | x 0.50–0.62 to 12.72–12.85; y usually 1.78–2.05 to 6.78–6.92 |
| Footer | x 0, bottom aligned, h 0.42–0.52 in |

Do not use a thin top rule as a substitute for the banner. Do not use a small page-number pill as a substitute for the footer.

## 3. Title lockup

- Default title face: Source Han Serif/Noto Serif CJK SC SemiBold or Bold.
- Default title color: primary navy; use red only for a selected phrase or contrast word.
- Default title size: 29–36 pt depending on length; preserve one line whenever possible.
- Number badge: navy rounded rectangle with white two-digit number, 0.06–0.10 in radius.
- Align badge and title optically on the same baseline. The badge should feel like the title's first glyph.
- Add a short 3–5 pt equivalent red rule beneath the badge/title start.
- Use the subtitle as a concise explanatory sentence, not a second competing headline.

Failure pattern: a floating rounded-square number, generic bold sans headline, tiny red line, and large empty top margin. Fix the whole lockup, not only the font.

## 4. Rounded component grammar

- Outer cards use 0.08–0.14 in radius. Use one consistent radius per deck.
- Use 0.75–1 pt borders in warm gray or navy tint.
- Header bands are 0.28–0.42 in high, navy or red, with white semibold text.
- When a header band must have only top corners rounded, construct it as a rounded rectangle and cover its lower rounding with the body fill, or use an equivalent clipped path. Do not leave a square top bar on a rounded card.
- Circular icon containers are usually 0.28–0.48 in, solid navy/red with white line icons.
- Internal padding: 0.12–0.20 in for compact modules; 0.20–0.28 in for major cards.
- Shadows are normally absent. Separation comes from borders, fills, alignment, and header color.

## 5. Table construction

The reference table is an editorial component, not a pasted spreadsheet.

1. Place a rounded white/warm card behind the table.
2. Add a navy title band or column-header band with white bold text.
3. Use compact 14–17 pt labels and rows sized to the content; avoid tall empty rows.
4. Use subtle warm zebra fills such as soft panel / canvas alternation.
5. Prefer light horizontal separators; use vertical lines only when they materially aid comparison.
6. Color only the decision value, exceptional result, or selected row in red.
7. If paired with a chart, align both cards to the same top and bottom; table 35–45%, chart 55–65%.
8. Add a bottom interpretation strip when the table itself does not make the conclusion obvious.

For multi-paper review tables, compress verbose cells into phrase-level findings and move nuance into speaker notes or paper-specific slides. A five-column full-width grid with paragraph cells should be a deliberate synthesis page, not the default table treatment.

## 6. Page families

Maintain the frame and title lockup while varying the center:

- Overview: 4–6 rounded modules with numbered corner flags, icon circles, short chapter/goal lines, and connectors.
- Definition: two explanatory cards plus one dominant chart/diagram card and a bottom memory strip.
- Comparison: matched navy/red cards, aligned dimensions, central or bottom synthesis.
- Formula/result: 3–4 compact metric cards above a table/chart pair; interpretation strip below.
- Process: 6–8 numbered rounded steps with alternating navy/red emphasis and a right-side rule/checklist.
- Risks/errors: 5–7 narrow rounded category cards with large icons and short copy.
- Conclusion: 3–5 claims in structured cards or a concise two-column synthesis; retain footer and title identity.

Avoid using the same three-card rectangle layout repeatedly. The reference gains coherence from the frame and gains energy from central-layout variation.

## 7. Calibration workflow

Before building the full academic deck:

1. Choose a representative dense slide containing at least two component types, ideally a table/chart, process, or comparison.
2. Build and render that one slide.
3. Compare side by side with the reference family at full size.
4. Check silhouette before details: top banner, footer, title scale, content fill, roundedness.
5. Check component morphology: bands, borders, icons, table rows, takeaway strip.
6. Only after it passes, freeze reusable helpers/tokens and generate the rest of the deck.

## 8. Regression failures to reject

- Thin navy hairline at the top instead of the banner.
- No footer color band, or only a gray rule and isolated page number.
- Generic sans title with weak title/section contrast.
- Number badge detached from the title.
- Square cards or spreadsheet cells.
- Plain full-width table with heavy gridlines and no rounded container.
- Excessive empty canvas on content slides.
- Repeated dashboard-style rectangles with little iconography or layout variation.
- Header/footer omitted because they were treated as optional without user instruction.
